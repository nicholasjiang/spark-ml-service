# spark-ml-service
开发spark 机器学习算法服务化 </br>
目前支持的算法：</br>
   als     （ALSAlg）</br>
   决策树   （DecisionTreeClassification）</br>
   FPG     （FPGrowthSD）</br>
   LDA     （LDASD）</br>
   线性回归  （LinearRegression）</br>
   逻辑回归  （LogisticRegressionLBFGS） </br>
   贝叶斯    （NaiveBayesSD）</br>
   随机森林   （RandomForestSD）</br>
   支持向量机  （SVM）</br>
